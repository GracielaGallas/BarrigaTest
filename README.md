# BarrigaTest
 Test application created to practice Cypress.
 Using https://barrigareact.wcaquino.me
 The test will have 3 levels: 
 - Functional tests
 - Backend tests (API)
 - Frontend tests
 
## test scope
 - Insert new account
 - Change account
 - Attempt to insert duplicate account
 - Insert transaction
 - Calculate balance
 - Remove a transaction
